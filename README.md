## 联想小新锐7000

-----[ 配置信息 ]-------------------------

CPU :i7-7700HQ

内存 : 8GBx2

硬盘 : 东芝XG3 256GB & 1TB 机械硬盘 5400RPM

显卡 ：NVDIA GTX1050(屏蔽) & Intel HD Graphics 630

声卡 ：ALC 235

有线网卡 ：RTL8168/8111

无线网卡 ：BCM94352z 联想版

-----[ 已知问题 ]-------------------------

1.无法睡眠

2.亮度调节最后最暗档位真的是最暗

3.HDMI无显示，不过不会重启

4.无法调节亮度请将CLOVER/kexts/Other/AppleBacklightInjector.kext放入L/E或S/L/E可修复亮度

5.有时候会无法启动，强制重启即可

-----[ 适用系统 ]-------------------------

macOS High Sierra 10.13 

macOS Mojave 10.14
